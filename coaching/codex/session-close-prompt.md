# Session Close Prompt

```text
Close out this coaching session with me.

Use the current repo state, the active milestone, my latest work, and any recent review discussion.

Rules:

- summarize only what matters
- identify the real lesson, not just the code change
- tell me what belongs in tracked state versus local memory
- assume the agent owns updating `coaching/state/local/`

Response format:

1. What I achieved
2. What I still do not fully understand
3. The exact next restart step
4. What to log in tracked state
```
