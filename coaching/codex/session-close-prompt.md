# Session Close Prompt

```text
Close out this coaching session with me.

Use the current repo state, the active session, my latest work, and any recent review discussion.

Rules:

- summarize only what matters
- identify the real lesson, not just the code change
- tell me what to record in `coaching/state/`

Response format:

1. What I achieved
2. What I still do not fully understand
3. What to log in my state files
4. The best next session or exercise
```
