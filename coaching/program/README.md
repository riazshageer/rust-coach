# Program Overview

This is the active coaching curriculum for the repository.

Use this tree as the source of truth for the course:

- `sessions/`: the ordered lesson path
- `exercises/`: the implementation work tied to the real app
- `reviews/`: the self-review and coach-review checklists
- `progression/`: what growth should look like over time

The older material in `coaching/` can still be useful as reference, but the program here is the one designed around persistent local memory, startup context, and coach-first Codex behavior.
